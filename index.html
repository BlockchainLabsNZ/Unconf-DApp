UnConf DApp

UnConf lets you create or participate in Unconferences

Commands

  • \createUnconf

    This command will deploy a new UnConf contract on the Blockchain.

    Params
    • Unconference name a short name for the UnConference (must be unique).

  • \register

    This will add you to the members' list of an existing UnConference.

    Params
    • Unconference name the name of the UnConference that you wish to join.
    • Alias an Alias that will be associated to your address for this UnConference.

  • \addTopic

    This will add a new topic to the UnConference.

    Params
    • Topic Name the name of topic to add to the list (must be unique per UnConference).

  • \voteForTopic

    This will add an upvote for an existing topic.

    Params
    • Topic Name the name of topic that you want to support.

  • \listTopics

    This will return a ranking with all of the topics and number of votes per topic.
